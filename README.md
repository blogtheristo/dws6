The EU mandates embodied carbon measurement for buildings >1,000 m¬≤ by 2028. Fit for 55 targets (55% emissions reduction by 2030) aren't aspirational‚Äîthey're compliance deadlines.

Buildings account for approximately 40% of CO‚ÇÇ emissions in the EU over their whole life cycle. Manufacturing contributes 24% of EU greenhouse gas emissions, while SMEs account for 63% of all European carbon emissions.

**DWS6 agents do.**

## Technical Architecture

### Hybrid AWS + Google + NVIDIA

- **Edge Layer**: AWS IoT Greengrass + NVIDIA Jetson Orin Nano (50 sites)
- **Core Platform**: Google Cloud Run + Groq API + LlamaStack
- **Client**: Chromebook Plus with Progressive Web App
- **Data Storage**: Supabase (hot) + AWS S3 (cold archive)
- **Cost**: $180/month after credits (86% cheaper than AWS-only)
